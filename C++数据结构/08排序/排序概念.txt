排序概念总结
1、排序是数据元素从无序到有序的过程
2、排序具有稳定性，是选择排序算法的因素之一
3、比较和交换是排序的基本操作
4、多关键字排序于但关键字排序无本质区别

选择排序逻辑
运用for 嵌套 for 内测初始化 j = i+1；创建一个变量（min）用来存储最小值或最大值下标  
内测循环 if (arr[i]<arr[min]) 判断获取最小值   min = j   最小值下标
内存循环结束 if (min != i) 就交换 min 和 i

插入排序
1、将无需序列插入到有序序列中
2、插入排序在什么情况下效率高
一、如果序列基本有序情况下
二、插入排序时候，数据序列比较少


希尔排序
先创建一个 变量用来记录分组的增量  先初始化等于元素总长 
增量算法   increasement /3 +1
分组排序完，最后合并后还需在排序一次，利用 do whele 判断增量 >1 
for循环先遍历每一组的第一个元素
嵌套for循环 遍历当前一组 
if 判断 arr[j]<arr[j+增量]  在for 排序


快速排序
分治法 ：大问题分解成各个小问题，对小问题求解，使得大问题得以解决


归并排序
归并排序思想： 将两个有序序列合并成一个有序序列

先把数据利用递归分组成有序序列  并加以判断退出添加  在合并
合并算法， 创建两组的起始和结束临时下标 start  end 、辅助内存temp
然后遍历并依次比较数据从两组的起始开始比较，(升序)小的值放入临时内存中
完成后再利用while 遍历一下两组序列是否还有数据 有继续放入临时内存
最后在把辅助空间数据覆盖原空间
